# MySpotifyWrapped

## Overview
This project analyzes personal Spotify streaming history to generate insights similar to a Spotify's own Spotify Wrapped.

## Current Statistics
- Top Artists and Songs
- Listening Trend Graphs
- Yearly Analyses
- Skipped and Picked Songs

## Data Source
This project uses Spotify Streaming History JSON exports. You can download your own data from your Spotify Account.

Example files used:
- Streaming_History_Audio_2017-2023_0.json
- Streaming_History_Audio_2023-2025_1.json

## Dependencies
- pandas
- numpy
- plotly
- matplotlib
- seaborn
